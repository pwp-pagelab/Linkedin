How to export as a LinkedIn-ready PDF (Chrome recommended):
  1. File → Print (or Ctrl/Cmd + P)
  2. Destination: Save as PDF
  3. Paper size: Custom → set Width: 28.58 cm and Height: 28.58 cm (= 1080 × 1080px square)
  4. Margins: None
  5. Scale: 100%
  6. Check "Background graphics" — required for colors and backgrounds
  7. Click Save

On LinkedIn: Start a post → click the document icon → upload the PDF. Each page becomes one swipeable slide.
